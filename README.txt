To run Percolation:
Run Percolation.class in bin folder
Enter input when prompted

Alternatively:
Open Eclipse
Load Percolation in as a project
Run it in Eclipse
Enter input when prompted

Ouputs:
If chosen output will print to command line
3 files will be produced
output.txt - visual representation of model
saturation.txt - each line is total acculmulated saturated cells
row_saturation.txt - each line is number of saturated cells in current line

WARNING
Printing to command line may be confusing it world width is set to value 
greater than width of your command line
